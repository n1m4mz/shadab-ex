# فرمت پیام تلگرام برای نرخ ارز

برای به‌روزرسانی نرخ‌های ارز در سایت، پیام خود را در کانال تلگرام به این **فرمت ساده** ارسال کنید:

## فرمت ساده (پیشنهادی)

```
date: 1403/11/16
gregorian: 2025/02/05

USD: 45000
EUR: 48000 | Euro | 🇪🇺
GBP: 55000 | British Pound | 🇬🇧
TRY: 1500 | Turkish Lira | 🇹🇷
IRR: 450 | Iranian Rial | 🇮🇷
AFN: 600 | Afghan Afghani | 🇦🇫
CAD: 33000 | Canadian Dollar | 🇨🇦
AUD: 30000 | Australian Dollar | 🇦🇺
```

## فرمت هر خط:

```
CURRENCY_CODE: RATE | NAME | FLAG
```

### مثال‌های مختلف:

**ساده‌ترین فرمت (فقط کد و نرخ):**
```
USD: 45000
EUR: 48000
```

**با نام ارز:**
```
USD: 45000 | US Dollar
EUR: 48000 | Euro
```

**کامل (با نام و پرچم):**
```
USD: 45000 | US Dollar | 🇺🇸
EUR: 48000 | Euro | 🇪🇺
GBP: 55000 | British Pound | 🇬🇧
```

## فیلدها:

### تاریخ (اختیاری)
- `date:` تاریخ شمسی (مثال: `date: 1403/11/16`)
- `gregorian:` تاریخ میلادی (مثال: `gregorian: 2025/02/05`)

### نرخ ارز (الزامی)
هر خط یک ارز با فرمت:
- **CURRENCY_CODE**: کد 3 حرفی ارز (مثال: `USD`, `EUR`) - **الزامی**
- **RATE**: نرخ ارز (مثال: `45000`) - **الزامی**
- **NAME**: نام ارز (مثال: `US Dollar`) - **اختیاری** (در صورت عدم وجود از کد ارز استفاده می‌شود)
- **FLAG**: ایموجی پرچم (مثال: `🇺🇸`) - **اختیاری** (در صورت عدم وجود از 💱 استفاده می‌شود)

## نکات مهم:

1. **فرمت ساده**: هر خط یک ارز، با دو نقطه (`:`) بین کد و نرخ
2. **جداکننده**: برای نام و پرچم از علامت `|` استفاده کنید
3. **کد ارز**: باید دقیقاً 3 حرف باشد (USD, EUR, GBP, ...)
4. **نرخ**: می‌تواند با کاما یا بدون کاما باشد (`45000` یا `45,000`)
5. **فاصله**: فاصله‌ها اختیاری هستند (`USD:45000` یا `USD: 45000`)

```
IRR: 450 | Iranian Rial | 🇮🇷

```

## مزایای فرمت جدید:

✅ **خیلی ساده**: فقط کد و نرخ، بدون نیاز به JSON  
✅ **راحت برای تایپ**: می‌توانید سریع در تلگرام تایپ کنید  
✅ **انعطاف‌پذیر**: می‌توانید فقط کد و نرخ بفرستید یا کامل با نام و پرچم  
✅ **بدون کش**: داده‌ها همیشه تازه هستند و نیازی به Ctrl+F5 نیست  

این فرمت بسیار ساده و کاربردی است! 🎉
